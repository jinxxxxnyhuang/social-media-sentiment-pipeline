# Social Media ETL & Sentiment Monitoring System

## Overview
This project builds an end-to-end data pipeline to collect, process, and analyse social media data from Mastodon.

## Features
- Automated data collection (cron jobs)
- Data cleaning and transformation (Python)
- Sentiment analysis using XLM-RoBERTa
- Data storage in Elasticsearch
- Interactive dashboards in Kibana
- Pipeline monitoring (data quality tracking)

## Tech Stack
Python, Elasticsearch, Kibana, Cron, Transformers

- Sentiment Analysis
- Pipeline monitoring
